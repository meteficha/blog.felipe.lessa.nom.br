Nenhuma dessas tags pode ser colocada dentro de si mesma. Por exemplo, você não deve fazer isto: "<em>Quero dar <em>MUITA ÊNFASE</em> aqui</em>"

Blocos

Você pode criar facilmente blocos como este aqui. Por exemplo, para criar este bloco basta usar<block type="explanation" title="Blocos">...</block>. Para criar um bloco sem título, use, por exemplo,<block type="exercise">...</block>

O tipo do bloco determina a sua cor. Por enquanto existem dois tipos de blocos:

Tipo de blocoAparência
"explanation"Rosa
"exercise"Verde

Note que em todos os casos não há acentos nos tipos dos blocos!

Imagens

Depois de uma imagem ter sido adicionada ao site, você pode embuti-la nas páginas. As imagens devem sempre ser embutidas dentro de tags <p>...</p>.

Para embutir a miniatura da imagem, basta ir para alista de imagens, escolher a imagem desejada e copiar a tag correspondente ao tamanho da imagem que você quiser.

Por exemplo, para embutir imagem de código 5 que você encontrou na lista de imagens de forma centralizada na página e sem texto em volta, você poderia escrever <p><img src=5 type=full></p>.

Tabelas simples

Para fazer uma tabela simples, use a tag <simpletable>...</simpletable>. Dentro dela coloque uma tag <row>...</row> por linha. A primeira das linhas vai ser o cabeçalho. Dentro de cada linha, use <br> para ir para a próxima coluna. Note que tabelas devem ser colocadas fora dos parágrafos <p>!

Por exemplo, suponha que você queira escrever a tabela abaixo:

NúmeroNome
23Joãozinho
31Maria
49Rafaela

Basta então escrever:

<simpletable>
  <row>Número<br>Nome</row>
  <row>23<br>Joãozinho</row>
  <row>31<br>Maria</row>
  <row>49<br>Rafaela</row>
</simpletable>

Tabelas complexas

Se você precisar quebrar a linha dentro de uma coluna usando <br> ou precisar colocar mais de uma linha dentro do cabeçalho, você precisará usar as tabelas complexas. Para fazer uma tabela complexa, use a tag <table> <thead>...</thead> <tbody>...</tbody> </table>. Dentro de <thead>...</thead> devem ser colocadas as linhas do cabeçalho e dentro de <tbody>...</tbody> o conteúdo da tabela.

Dentro de uma tag <thead>...</thead> ou <tbody>...</tbody> são colodas linhas <tr>...</tr>, e dentro das linhas são colocadas colunas<td>...</td>. Dentro da coluna você deve escrever como se estivesse dentro de um parágrafo.

Por exemplo, para fazer a mesma tabela do exemplo anterior, escreva:

<table>
  <thead>
    <tr>
      <td>Número</td>
      <td>Nome</td>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>23</td>
      <td>Joãozinho</td>
    </tr>
    <tr>
      <td>31</td>
      <td>Maria</td>
    </tr>
    <tr>
      <td>49</td>
      <td>Rafaela</td>
    </tr>
  </tbody>
</table>

Poesias

Para escrever uma poesia, ponha cada estrofe em um parágrafo <p>...</p> e entre versos coloque <br>. Por exemplo:

<p>Amor é fogo que arde sem se ver (Camões)</p>

<p>Amor é fogo que arde sem se ver;<br>
É ferida que dói e não se sente;<br>
É um contentamento descontente;<br>
É dor que desatina sem doer;</p>

<p>É um não querer mais que bem querer;<br>
É solitário andar por entre a gente;<br>
É nunca contentar-se de contente;<br>
É cuidar que se ganha em se perder;</p>

<p>É querer estar preso por vontade;<br>
É servir a quem vence, o vencedor;<br>
É ter com quem nos mata lealdade.</p>

<p>Mas como causar pode seu favor<br>
Nos corações humanos amizade,<br>
Se tão contrário a si é o mesmo Amor?</p>
